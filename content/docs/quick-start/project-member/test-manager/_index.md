# 测试管理
---

Choerodon为您带来持续交付的高效开发的同时也为您提供了测试管理来优化软件测试的速度和质量，让测试能够跟随上开发生命周期的步伐。

测试管理为用户提供敏捷化的持续测试工具，包括`测试用例管理`、`测试循环`、`测试分析`等，可以有效地提高软件测试的效率和质量，提高测试的灵活性和可视化水平，最终减少测试时间，让用户将主要精力放到软件功能构建上。
