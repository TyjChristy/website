# 项目设置

项目设置主要设置问题编码的项目前缀。

## 菜单路径

* 菜单层次：项目层
* 菜单路径：大规模敏捷>设置>项目设置
* 默认角色：项目所有者

## 操作步骤

填写`项目编码`

保存后，此项目上的所有问题编号的前缀将同步修改
    {{< note >}}
`项目编码`会校验是否与其他项目同名，若与其他项目同名则检验不通过
{{< /note >}}


{{< note >}}
项目编码用于问题编号的前缀，如无特殊需求，不建议频繁修改
{{< /note >}}
