# 测试执行

通过本章您将了解到在Choerodon中如何执行测试，查看测试进度。

![](/img/docs/user-guide/test-management/test-execute/test-execute.png)


1. 对执行进行筛选：分为`我的执行`、`所有执行`。
2. 收起测试执行用例树
3. 测试执行的阶段执行进度
4. 测试执行的详细信息
5. 对测试用例进行快速搜索：分为执行方、被指定人
6. 点击此按钮可以进入到测试执行页面，可以对测试步骤的结果进行记录，查看执行的详细信息及变更状态。
7. 直接将执行状态置为通过
8. 直接将执行状态置为失败
