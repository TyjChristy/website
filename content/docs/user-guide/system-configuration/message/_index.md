# 消息中心

<h2 id="1">介绍</h2>

消息中心分为邮件、短信、站内信3大类别。它包含了激发系统与用户交互的触发点类型，不同触发点在触发之后给用户反馈的消息模板，发送邮箱和短信的服务器配置信息。在消息中心，您可以管理您的不同类型的模板，发送消息的服务器配置。

- 在左侧导航栏里，点击`消息中心`进入消息中心。
