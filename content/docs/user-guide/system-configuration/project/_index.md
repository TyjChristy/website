## 介绍
  
本页面介绍了项目层的项目角色分配及项目信息的介绍。想要进行这些操作，您必须是一个项目的项目管理员，您可以在`组织设置下`的`项目管理`来创建项目。

- 在项目层界面中，进入“项目设置”界面。
- 需要了解如何[创建项目](..//tenant/project)。
