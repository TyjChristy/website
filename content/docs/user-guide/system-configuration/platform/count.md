# 平台统计

平台统计是对用户在平台的访问情况的汇总统计。菜单点击统计是对系统内的目录以及子菜单都有多少人点击数量的统计，同时也可以对比出不同菜单的点击次数。

- **菜单层次**：全局层
- **菜单路径**：平台设置 > 平台统计
- **默认角色**：平台管理员

## 菜单点击统计

- 通过图表您可以了解平台、组织、项目3个层级的所有菜单30天内每天点击情况。您可以切换层级，了解不同层级菜单的点击量。您也可以切换30天内的任意时间范围来了解不同时间段菜单的点击量。
